# Search Algorithms (JavaScript)

Implementations of search algorithms and related data structures in JavaScript.

## Files

- **[binary_tree.mjs](./binary_tree.mjs)**: Binary Tree implementation and traversal.
- **[count_pair.mjs](./count_pair.mjs)**: Algorithm to count specific pairs in a collection.
- **[find_most_common_url.mjs](./find_most_common_url.mjs)**: Find the most frequent URL in a list.
- **[singly_linked_list.js](./singly_linked_list.js)**: Singly Linked List implementation.
- **[maximum-call-stack-overflow.js](./maximum-call-stack-overflow.js)**: Demonstration of stack overflow limits.
