# Codility Challenges (JavaScript/TypeScript)

Solutions to challenges from the Codility platform implemented in TypeScript.

## Files

- **[binary-gap.ts](./binary-gap.ts)**: Find the longest sequence of zeros in binary representation of an integer.
- **[cyclic-rotation.ts](./cyclic-rotation.ts)**: Rotate an array to the right by a given number of steps.
- **[frog-jump.ts](./frog-jump.ts)**: Count the minimal number of jumps from position X to Y.
- **[odd-occurrences-in-array.ts](./odd-occurrences-in-array.ts)**: Find value that occurs in odd number of elements.
- **[code-challenge-quinto-andar.ts](./code-challenge-quinto-andar.ts)**: Specific coding challenge implementation.
