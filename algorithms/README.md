# Algorithms

This directory contains implementations of various algorithms categorized by their type and source.

## Categories

- **[basics/](./basics/)**: Fundamental logic and basic loop exercises.
- **[codility/](./codility/)**: Solutions to challenges from the [Codility](https://app.codility.com/programmers/lessons/) platform.
- **[search/](./search/)**: Searching algorithms like Binary Search, and related structures like Binary Trees.
- **[sort/](./sort/)**: Classic sorting algorithms including Bubble Sort and Insertion Sort.

## Languages Used

- JavaScript (ES Modules)
- TypeScript
- Python (Jupyter Notebooks)
